Using Data Type

Variable ေၾကျငာပုံ

(Data type) (Variable Name) = (Data);

ပထမအေနနဲ႕  Data Type ေတြအေၾကာင္းေျပာပါမယ္ ။

* String

String data type က ေတာ့ စာေၾကာင္း ေတြကုိ သိမ္းရန္ျဖစ္ပါတယ္။

ဒီလုိသိမ္းတဲ႕ေနရာမွာ မိမိ သိမ္းခ်င္တဲ႕စာသားေတြကုိ " နဲ႕ "
 ၾကားအတြင္းထည့္ေပးရပါမယ္ ။

/**************************/
Code:
Example :

String name = "ZMO";
/**************************/

ဒီ Example မွာ name ဆုိတဲ႕ variable ကုိ String data Type ေၾကျငာလုိက္ပါတယ္ ။ ျပီးေတာ့ "ZMO" ဆုိတဲ႕ စာသားကုိသိမ္းလုိက္ပါတယ္ ။

* char

chare data type ကေတာ့ စကာလုံးတစ္ခုျခင္းဆီကုိသိမ္းတဲ႕ေနရာမွာသုံးပါတယ္ ။

/**************************/
Code:
Example :

char firstWord = 'A';
/**************************/

မွတ္ခ်က္ ။ single character (သုိ႕) စကားလုံးကုိသိမ္းတဲ႕ေနရာမွာ ' နဲ႕ ' အတြင္းသိမ္းေပးရပါမယ္။ ( ' ) ကုိ  Single code လုိ႕ ေခၚျပီး ( " ) ကုိ Double

code လုိ႕ေခၚပါတယ္ ။

* integer

integer data type ကေတာ့ ကိန္းျပည့္ေတြကုိသိမ္းတဲ႕ေနရာမွာအသုံးျပဳပါတယ္ ။

/**************************/
Code:
Example :

int number = 1;
/**************************/

မွတ္ခ်က္ ။ ကိန္းျပည့္ေတြကုိသိမ္းတဲ႕အခ်ိန္မွာ ( " ) Double code မထည့္ရပါဘူး ။ "123" နဲ႕ 123 မွာ "123" ကစာသားျဖစ္ျပီး String data ျဖစ္ပါတယ္ ၊

123 ကေတာ့ integer data ျဖစ္ပါတယ္ ။ Integer data ကသာ
ေပါင္း + နုတ္ - ေျမွာက္ * စား / ျပဳလုပ္နုိင္ပါမယ္ ။

* boolean

boolean data type ကေတာ့ True အမွန္ False မွား တန္ဖုိးတစ္ခုကုိ သိမ္းခ်င္တဲ႕ေနရာမွာအသုံးျပဳပါတယ္ ။

/**************************/
Code:
Example :

boolean isFinish = true;
/**************************/

မွတ္ခ်က္ ။ true false တန္ဖိုးနွစ္ခုသာသိမ္းနိုင္ပါတယ္ ။ true false က String မဟုတ္ပါဘူး ။ True False TRUE FALSE စသည္ေရးမရပါဘူး Error

တက္ပါမယ္ ။ true false ဆုိသည့္ samll letter ျဖင့္သာေရးပါ ။

* double

double data type ကေတာ့ ဒသမကိန္းဂဏန္းေတြကုိ သိမ္းတဲ႕ေနရာမွာ အသုံးျပဳပါတယ္ ။ Integer မွာ သိမ္းမရပါဘူး ။

/**************************/
Code:
Example :

double rate = 1.5;
